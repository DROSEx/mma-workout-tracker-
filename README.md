# MMA Workout Tracker

**Progressive Web App (PWA)** for logging MMA-inspired workouts. Built as a mobile-first tool while transitioning from concreting to AI/systems engineering and maintaining training consistency at Fitness First Melbourne clubs.

## What It Does
- Follows a 6-day MMA athlete training plan (strength, conditioning, HIIT, striking via Boxing Circuit / BODYCOMBAT classes).
- Log actual sets, reps, weights, notes, gym used (Richmond Platinum, QV Platinum, Flinders St Platinum, etc.), and date.
- Fully offline — all data saved privately on your device using localStorage.
- Installable as a home-screen app on Android (add to home screen from Chrome).

## Features
- Pre-loaded 6-day plan tailored for a 33-year-old with MMA/kickboxing background (non-pro).
- Workout history and basic progress summary.
- Gym selector with common Melbourne Fitness First locations.
- Lightweight design that runs smoothly on budget Android devices like the Oppo A38.

## Tech Stack
- Vanilla HTML, CSS, JavaScript
- LocalStorage for persistence
- Progressive Web App (PWA) — works offline after first load

## How to Use
1. Open `index.html` in any modern browser.
2. Add to home screen for a native app-like experience.
3. Log your workouts daily and track progress over time.
